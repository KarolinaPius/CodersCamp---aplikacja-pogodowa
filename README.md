# WeatherApp 

Application displays weather forecast for searched location.
Forecast includes:
  - current detailed weather
  - 3 day detailed forecast
  - hourly forecast
  - daily forecast
  
  Click [HERE](https://karolinapius.github.io/CodersCamp---aplikacja-pogodowa/) for live demo.
  
  Data comes from [OpenWeather's API](https://openweathermap.org)
